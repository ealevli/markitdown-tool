How it works
Three steps, zero friction
No account, no upload limits, no waiting. Your files are processed on your machine and never stored.
1

Drop your file

Drag & drop any supported file, click to browse, or paste a URL. Works with PDFs, Office files, images, web pages and more.

2

Convert instantly

Powered by Microsoft's open-source MarkItDown library, conversion happens server-side in milliseconds. Files are deleted immediately after processing.

3

Copy or download

Copy to clipboard with one click or download the .md file. See character count, line count and token estimate.

Features
Everything you need, nothing you don't
Built for developers, writers, and anyone feeding content into AI models.
🔒

100% local processing

Your files are converted on your own machine. Nothing is uploaded to any server. Not even temporarily.

🔗

URL to Markdown

Paste any public URL — web page, PDF, news article — and get clean Markdown back instantly.

📁

Batch conversion

Select multiple files at once. They're converted in order and you can switch between results with a single click.

📥

Download as .md

Copy to clipboard or download the result as a proper .md file, named after your original document.

🪙

Token estimate

See an approximate token count alongside character and line counts — useful when preparing context for LLMs.

⚡

No size headaches

Files up to 50 MB are supported. The tool checks size before upload so you never wait for a rejection.
